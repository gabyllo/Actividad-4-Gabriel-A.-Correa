Tema 2. Define los elementos que componen la estructuración de una página web:

1. Contenedor (wrapper):

Un contenedor (wrapper) es un programa o código que rodea otros componentes de un programa

2. Cabecera (header):

La cabecera es la parte superior de una página web, donde se colocan los elementos más importantes para la navegación y la marca.

3. Contenido (content):

El contenido de una página web es la información que se presenta en ella, y que puede ser de texto, imágenes, audio, video, programas, infografías, entre otros.

4. Menú (menu):

un menú es una lista de contenido desordenada que se define con la etiqueta menu. Para crear elementos de menú, se utiliza la etiqueta menu junto con la etiqueta li

5. Pie (footer):

Es la sección de una página web que se ubica en la parte inferior del documento.

6. Lateral (sidebar):

Elemento que se encuentra al lado del contenido de una página web y que sirve para mostrar información adicional.

7. Etiqueta div:

Elemento de bloque que se utiliza para agrupar secciones de contenido de una página web. Es un contenedor "puro" que no tiene significado semántico, pero que permite crear y delimitar áreas para ser formateadas con CSS.
